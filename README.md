<h1>Explorador de Licencias Open Source</h1>
<p><h2>Compara y analiza las principales licencias de código abierto</h2></p>
<p>Esta aplicación esta desarrollada con React/TS utilizando las librerias estandar y agregando lucide para los iconos y tailwindcss para todo lo referente a la interfaz de usuario </p>
<p>La idea era tener un referente rapido y sencillo para visualizar los diferentes tipos de licencias opensource que actualmente son utilizadas en los proyectos 
  , el grado de libertad otorgado, los pro y los contra de cada una de las licencias, donde utilizarla y algunos ejemplos de proyectos realizados baja esa licencia.</p>
